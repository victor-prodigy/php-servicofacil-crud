# Inicializar

1. Executar `Apache` e `MySQL` no XAMPP Control Panel
2. Acessar http://localhost/php-servicofacil-crud-incompleto/

# DB

- /lib/db.sql -> é o codigo para criar o banco de dados e as tabelas
- /lib/seed.sql e /lib/seed-admin.sql são para inserir dados nas tabelas

1. o comando para criar o banco de dados é `php php/create-database.php`
2. o comando para criar as tabelas é `php php/setup-database.php`
